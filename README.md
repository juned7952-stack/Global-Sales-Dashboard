🌍 Global Orders and Sales Dashboard
This Power BI project provides an interactive Global Orders and Sales Dashboard that visualizes key sales insights across different regions, products, and time periods.
It helps users understand overall performance, identify trends, and make data-driven decisions.

📊 Project Overview
The Global Orders and Sales Dashboard is designed to analyze worldwide sales data and present it in an easy-to-understand format.
The dashboard allows users to explore metrics such as:

Total Revenue and Profit
Regional Sales Distribution
Top-Selling Products and Categories
Monthly/Yearly Sales Trends
Customer Segment Analysis
KPIs and Performance Indicators
🎯 Objectives
To analyze global sales patterns and trends
To identify high-performing regions and products
To compare sales performance across different time periods
To visualize insights for better business decision-making
🧰 Tools & Technologies
Power BI Desktop
Data Cleaning & Transformation: Power Query Editor
Data Modeling: Relationships between multiple tables
Visualizations: Line Charts, Bar Charts, Maps, Cards, and Slicers
